🔑 CREDENTIALS YOU NEED (Get from IMAGE Firebase Project)
═══════════════════════════════════════════════════════════════════════════════
1. PROJECT ID
   Where: Project Settings → General (top section)
   Format: backtoyou-images-xyz123
   Example: backtoyou-images-abc123
2. API KEY
   Where: Project Settings → API keys (copy Android key)
   Format: AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
   Example: AIzaSyDc1234567890abcdefghijklmnopqrst
3. APP ID
   Where: google-services.json (search for "mobileSdkAppId")
   Format: 1:123456789:android:abcdef1234567890
   Example: 1:123456789012:android:abcdef1234567890
4. STORAGE BUCKET
   Where: Project Settings → General OR Storage console
   Format: backtoyou-images-xyz.appspot.com
   Example: backtoyou-images-abc123.appspot.com

📝 STORAGE RULES (Copy & Paste into IMAGE project)
═══════════════════════════════════════════════════════════════════════════════
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /images/{allPaths=**} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.resource.size < 50 * 1024 * 1024;
      allow delete: if request.auth != null;
    }
  }
}
